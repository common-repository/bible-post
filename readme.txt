== Description ==

Es una necesidad para todo conocedor de lo que Dios quiere en  nuestras vidas el impartir el conocimiento que nos fue dado. Es por ello
que este plugin se creo a necesidad de usar como medio de proclamar el evangelio a toda criatura.

Con este Plugin podras:

* Incluir un versiculo favorito (Widget)
* Incluir un pasaje biblico en una publicacion.
* ShortCode

Leer **[Mas > Descripcion](http://www.hyno.ok.pe/wp-plugins/bible-post/bible-post-2-0/)**

== Installation ==

1. Descomprimir el plugin a la carpeta `wp-content\plugins\`.
2. Activarlo entrando a "administracion de Plugins" (AdminPanel/Plugins).
3. Podras usar el Plugin con el ShortCode o con el boton en editor de Wordpress:
4. Crear una nueva pagina llamada "Biblia" y asignarme como Template "La Santa Biblia(BiblePost)"
'(esa pagina solo sera usada por las referencias - Debe conocer el ID de la pagina para ser usada en la confuguracion)'

Leer **[Mas > Instalacion](http://www.hyno.ok.pe/wp-plugins/bible-post/bible-post-2-0/#instalacion)**
